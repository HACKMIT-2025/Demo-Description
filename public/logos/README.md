# Logo Directory

Place your downloaded logo files here. Recommended format:
- PNG or SVG format
- Transparent background preferred
- Size: at least 200x200px for quality

## Example filenames:
- modal-logo.png
- cerebras-logo.png
- windsurf-logo.png
- vercel-logo.svg
- anthropic-logo.png

## How to use in code:
```html
<img src="/logos/modal-logo.png" alt="Modal" class="w-8 h-8 object-contain" />
```
